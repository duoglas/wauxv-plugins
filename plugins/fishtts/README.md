# FishTTS

直接调 fish.audio `/v1/tts`，不经过 tiax.pw 中转。带 UI 弹窗（可滚动 / 搜索 / 试听 / 切换默认音色）。

> v1.2：补 4 个**真·低音炮**（磁性男声 / 低沉气泡音(ASMR) / 油腻气泡男5 / 气泡带点骚），共 **40 个预置音色**。
> v1.1：补 6 个男声（青叔 / 沉稳大叔 / 清朗少年 / 活力男 / 中二 / AI 名人）。

## 命令

| 命令 | 说明 |
|---|---|
| `#f 文本` | 用默认音色合成语音 |
| `#f N 文本` | 用第 N 号音色合成（编号见 `#fls`） |
| `#fls` / `#flist` / `#f列表` / `帮助 fish` | 弹窗：列出所有预置音色 + 搜索 + 一键试听 + 切换默认 |
| `#fset N` | 设默认音色为第 N 号 |
| `#fid <fish_id> 文本` | 用任意 fish.audio reference_id 临时合成 |
| `#fkey` | 弹窗设置 fish.audio API key |
| `#fkey <token>` | 行内直接设置 key（一行搞定） |

> 还可以在 WAuxiliary 主面板的插件列表里点 **FishTTS 的设置入口**，等同 `#fls` 弹窗。
